Job Description
We’re looking for a Junior Backend Developer to join our dynamic team. This is an exciting opportunity for someone eager to grow in a collaborative environment and help build reliable backend systems for scalable web and mobile applications. As a key member of the development team, you will work alongside experienced developers, gain hands-on experience with industry-leading technologies, and contribute to the success of our products.

Key Responsibilities
Assist in designing and building RESTful APIs and microservices for web and mobile applications
Collaborate with the frontend team to ensure seamless integration of APIs and features
Maintain and optimize existing backend infrastructure, ensuring high performance and availability
Write clean, maintainable, and efficient code, following best practices and coding standards
Troubleshoot, debug, and improve backend applications
Participate in code reviews, provide constructive feedback, and learn from senior developers
Assist in database design and management (PostgreSQL, MongoDB, etc.)
Stay up-to-date with new backend development trends and technologies
Requirements
Bachelor’s degree in Computer Science, Software Engineering, or a related field (or equivalent experience)
Familiarity with backend development using Node.js, Express, or similar frameworks
Basic knowledge of REST APIs and microservices architecture
Experience with databases such as MongoDB, PostgreSQL, or MySQL
Understanding of version control systems (Git)
Strong problem-solving skills and attention to detail
Eagerness to learn and adapt to new technologies
Preferred Skills
Experience with TypeScript
Familiarity with cloud platforms like AWS, Google Cloud, or Azure
Exposure to CI/CD pipelines and testing tools
Knowledge of containerization tools (e.g., Docker)
